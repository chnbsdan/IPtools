什么是 DNS 泄漏？为什么需要关注？(点击展开)

DNS 泄漏 (DNS Leak)是指在您使用代理或 VPN 时，您的设备仍然将域名解析请求发送给本地网络提供商（ISP）的 DNS 服务器。

为什么值得关注？

  • 隐私泄露：即使您的访问流量被代理软件加密，本地 ISP 依然可以通过 DNS 查询记录知道您访问了哪些网站。
  • 访问受限：如果请求未通过代理节点转发，您可能会遇到 DNS 污染，导致获取到虚假的网站 IP 从而无法访问外网。

理想安全状态：代理开启时，您的出口 IP 为国外，且您的 DNS 请求也应由国外的 DNS 服务器负责解析（即 IP 与 DNS 均为国外地区）。
